Arcade Box

Minigame Collection in Python

Home Projects About Me

Description:

Arcade Box is a Python-based arcade game program that offers a variety of classic games to entertain players. With a simple text-based interface, players can choose from a selection of games including Russian Roulette, Rock Paper Scissors, Guess the Number, and Tic Tac Toe. The program also features a scoreboard to track players' performance across multiple sessions.

Features:

  1. Game Selection: Players can choose from a menu of available games, each offering a unique gaming experience.
  2. Russian Roulette: Experience the thrill of a deadly game of chance where luck determines survival.
  3. Rock Paper Scissors: Test your strategy and intuition in the classic hand gesture game against the computer.
  4. Guess the Number: Try to guess the secret number within a specified range to win.
  5. Tic Tac Toe: Play the timeless game of Tic Tac Toe against a friend or the computer.
  6. Scoreboard: View and track your performance across different games with a built-in scoreboard feature.

How to Play:

  1. Launch the Arcade Box program.
  2. Choose a game from the menu by entering the corresponding number.
  3. Follow the on-screen instructions for each game to play.
  4. After playing, view your scores on the scoreboard to track your progress.

Purpose:

Arcade Box aims to provide users with a nostalgic gaming experience reminiscent of classic arcade games. With its simple interface and diverse game selection, it offers entertainment for players of all ages.

Technologies Used:

  • Python programming language
  • Text-based user interface
  • Randomization algorithms for game mechanics
  • Basic file handling for scoreboard functionality

Future Enhancements:

  1. Adding more games to the arcade library.
  2. Implementing graphical user interface (GUI) for a more interactive experience.
  3. Integrating multiplayer functionality for select games.
  4. Enhancing the scoreboard feature with additional statistics and player profiles.

Arcade Box brings the joy of arcade gaming to your fingertips with a collection of timeless classics, ensuring endless hours of fun and entertainment for players of all skill levels.
